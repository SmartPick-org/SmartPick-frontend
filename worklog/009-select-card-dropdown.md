# Worklog: Select card dropdown

## 왜 이렇게 했나요
- 기존 카드 선택은 드롭다운 형태여야 한다는 요구사항을 반영했습니다.
- 긴 카드 리스트 대신 드롭다운으로 선택해 화면이 단순해집니다.

## 무엇을 수정했나요
- 카드 선택 UI를 리스트에서 드롭다운으로 변경했습니다.
- 선택된 카드의 요약 정보(카드사/연회비/실적)를 표시했습니다.
- 테스트도 드롭다운 선택 방식으로 수정했습니다.

## 확인 방법
- 실행 경로: `/select-card`
- 기대 동작: 드롭다운에서 카드를 선택하면 하단 요약이 표시되고, 다음 버튼이 활성화됩니다.

## 참조 주석
