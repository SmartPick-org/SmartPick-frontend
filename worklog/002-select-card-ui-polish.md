# Worklog: Select card UI polish

## 왜 이렇게 했나요
- 카드 선택 화면은 사용자가 가장 먼저 상호작용하는 입력 단계라서, 정보 구조가 명확해야 합니다.
- 검색/선택 흐름이 눈에 잘 들어오도록 헤더와 리스트 구성을 정리했습니다.

## 무엇을 수정했나요
- 헤더에 단계 라벨과 설명을 추가해 현재 위치를 명확히 했습니다.
- 검색 영역을 강조하고, 검색 결과 수를 바로 보이게 했습니다.
- 카드 리스트를 더 큰 카드 형태로 정리하고 선택 상태를 강조했습니다.
- 선택된 카드 요약을 하단에 표시했습니다.

## 확인 방법
- 실행 경로: `/select-card`
- 기대 동작: 카드 선택 후 하단에 선택된 카드명이 표시되고, 다음 버튼이 활성화됩니다.

## 참조 주석
- 카드 리스트: 여러 선택지 중 하나를 고르기 쉽게 정렬한 UI 목록
- 상태 강조: 선택/비선택을 시각적으로 구분하는 표현 방식
