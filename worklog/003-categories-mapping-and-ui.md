# Worklog: Categories mapping and UI

## 왜 이렇게 했나요
- 서버 통신은 영문 Enum, 화면 표시는 한글이어서 매핑이 필요합니다.
- 카테고리 선택 화면은 정보 밀도가 높아지기 쉬워 구조를 정리했습니다.

## 무엇을 수정했나요
- 카테고리 메타/매핑 테이블을 추가했습니다. (한글 라벨 ↔ 영문 Enum)
- 카테고리 카드에 설명 문구를 넣고 레이아웃을 3열 카드형으로 정리했습니다.
- 선택 시 영문 Enum으로 변환해 상태에 저장하도록 변경했습니다.

## 확인 방법
- 실행 경로: `/categories`
- 기대 동작: 2~5개 선택 후 다음 버튼 활성화, 선택 시 영문 Enum이 저장됨

## 참조 주석
- Enum: 서버에서 미리 정의해 둔 값 목록(예: Food, Traffic)
- 매핑: 서로 다른 표기(한글/영문)를 연결하는 표
